#### [致敬 React: 为 Vue 引入容器组件和展示组件](https://juejin.im/post/5ae9a5545188256709610635)
    vue组件在和vuex store配合使用时，容易使组件耦合，导致脱离当前的项目很难复用。
    如果把组件分为容器组件和展示组件，容器组件与vuex配合，然后通过props把数据传给展示组件，这样减少了一定的耦合。
